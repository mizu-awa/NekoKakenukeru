# ねこかけぬける

ねこの各部位を個別に操作して障害物を避けるブラウザゲームです。

## 遊び方

ねこは自動で右に走り続けます。障害物にぶつかるとゲームオーバーです。
各部位を適切なタイミングでジャンプさせて、ゴールまで走り抜けましょう。

## ステージ構成

全10ステージ。部位数・速度・障害物密度が段階的に上昇します。ステージ8〜10は **Very Hard** です。

| ステージ | 部位数 | 速度 | 障害物数 | 難易度 |
|----------|--------|------|----------|--------|
| Stage 1  | 3      | 1.4  | 4        | Easy   |
| Stage 2  | 3      | 1.5  | 6        | Easy   |
| Stage 3  | 3      | 1.55 | 8        | Normal |
| Stage 4  | 4      | 1.5  | 10       | Normal |
| Stage 5  | 4      | 1.6  | 12       | Hard   |
| Stage 6  | 5      | 1.6  | 15       | Hard   |
| Stage 7  | 5      | 1.7  | 18       | Hard   |
| Stage 8  | 5      | 1.8  | 24       | **Very Hard** |
| Stage 9  | 6      | 1.8  | 26       | **Very Hard** |
| Stage 10 | 6      | 1.9  | 31       | **Very Hard** |

- Stage 4・9 で部位が増加（4パーツ、6パーツ）
- Stage 10 は最大6部位・最高速で31個の障害物をクリアするギリギリの難易度

## キー操作

### ゲーム操作

| キー | 動作 |
|------|------|
| `A` | 部位1（尻尾側）をジャンプ |
| `S` | 部位2をジャンプ |
| `D` | 部位3をジャンプ |
| `F` | 部位4をジャンプ（Stage 4以降） |
| `G` | 部位5をジャンプ（Stage 6以降） |
| `H` | 部位6をジャンプ（Stage 9以降） |
| `Space` | 全部位を一斉にジャンプ |
| `Enter` / `Space` | スタート・次へ・リトライ |

### 開発者向けキー

| キー | 動作 |
|------|------|
| `P` | **オートプレイ** のオン/オフ切り替え。AIが自動でジャンプを判断してプレイします。画面右上に `AUTO` と表示されます。 |
| `V` | **ステージ検証** を実行。全ステージをヘッドレスシミュレーションし、各ステージがクリア可能かどうかを判定してコンソールに出力します。結果は画面右上にも表示されます（✓ / ✗）。 |

## モバイル対応

画面下部のボタンで各部位をタップしてジャンプできます。
`ALL` ボタンで全部位を一斉にジャンプします。

## ファイル構成

```
nekokakenukeru/
├── index.html
├── game.js          # ゲームロジック本体
├── config.json      # ゲームパラメータ・ステージ定義
├── locales/
│   ├── ja.json      # 日本語テキスト
│   └── en.json      # 英語テキスト
└── img/
    ├── head.png     # ねこの頭スプライト
    ├── body.png     # ねこの胴体スプライト
    └── tail.png     # ねこの尻尾スプライト
```

## ブラウザで開く

ローカルサーバーを立てて `index.html` を開いてください。

```bash
npx serve .
# または
python -m http.server
```
